# Sentura Task

This repository contains the full-stack application divided into a backend and a frontend.

## Project Structure

- **`backend/`**: A Java Spring Boot application providing the REST API.
- **`frontend/`**: A React application set up with Vite.

## Getting Started

### Backend
1. Navigate to the `backend/` directory.
2. Ensure you have Java and Maven installed.
3. Run the Spring Boot application using Maven:
   ```bash
   mvn spring-boot:run
   ```

### Frontend
1. Navigate to the `frontend/` directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
